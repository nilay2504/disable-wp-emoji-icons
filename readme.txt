== Description ==

This plugin disable WordPress emoji icons ( from Post, Page, Comments and Email content ) and removes the extra JS code and Styles which is added by default in WordPress. This also helps to increase page load speed. 

If you have any question do let me know on gr8nilay@gmail.com


== Installation ==

1. Extract the zip file into your /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
1. On activation it will hide all the theme updates
